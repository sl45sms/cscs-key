The default launch mode opens this UI inside its own desktop window. Use --browser to fall back to a normal browser tab, or --headless to run only the local server. Override the detected binary with CSCS_KEY_BIN=/path/to/cscs-key or --bin when launching the UI.

Sign Existing Key

List Certificates

Fetch the certificates attached to the authenticated CSCS account.

Revoke Certificates

Revoke one or more certificate serial numbers, or revoke every active certificate.
